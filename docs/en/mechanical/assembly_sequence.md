# Assembly Sequence

This section introduces the robot assembly sequence.

## Notes

First assemble the robot body together with all motors and the IMU mounted on it. After that, the remaining parts can be installed one serial stage at a time.

1. Make sure the IMU is installed.
2. Pay attention to the assembly order. Some steps can be swapped, but some cannot. It is recommended to follow the order given in this document.
3. Pay attention to the bolt specifications used, including both diameter and length.
4. Positioning pins must be installed for the joint modules.
