# 装配顺序
本节介绍机器人装配顺序。

## 注意事项
先装机器人身体，以及身体上所有电机、IMU。后面逐个串联安装就可以了。

1. IMU一定要记得装
2. 注意安装顺序，部分顺序可以倒置，但是部分顺序不可倒置，建议按文件中顺序装配
3. 注意使用的螺栓规格（直径和长度）
4. 关节模组要装配定位销钉
