# 硬件与软件映射

本页用于梳理机器人硬件模块与软件模块之间的对应关系。

## 建议覆盖内容

- 传感器到驱动节点的映射
- 控制板与上位机的职责边界
- 执行器接口与控制循环的关系
- 仿真模型与实物部件的命名一致性
