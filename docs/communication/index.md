- 时间同步
- 示例代码

## 为什么需要通信框架

在一个机器人系统中，不同模块往往运行在不同进程、不同线程，甚至不同硬件上，例如：

- 传感器采集
- 状态估计
- 运动控制
- 上位机可视化
- 日志系统

通信框架的作用是为这些模块提供统一的数据交换机制。
